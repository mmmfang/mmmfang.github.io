What does this app do?

This app tracks the elevators and escalators that are out of service in the NYC subway system, which is run by the MTA. Information is updated every five minutes.


Key Features
  • Built with MEAN Stack - MongoDB, AngularJS, Express, Node.js
  • Responsive to different screen sizes
  • Uses external API - XML feed
  • Deployed on Heroku

    • Next Step - Mobile App

    I'm using Ionic, a hybrid mobile framework based on AngularJS, to deploy this web application as a mobile app for IOS and Android. Ionic wraps around Cordova and provides a UI library that makes the app look like it was natively built.

    User Stories/ Wireframe

    The target audience is people with limited mobility, seniors, those with strollers, or anyone who just doesn't feel like taking the stairs! It will help someone who can't walk up multiple flights of stairs - they will know to avoid the station before struggling up entry stairs, paying their fare...only to encounter a broken escalator or elevator!
