What does this app do?

This app reports which elevators and escalators are out of service in the NYC Subway System, which is run by the MTA. Information is accurate - updated every five minutes.


Key Features
  • Built with MEAN Stack - MongoDB, AngularJS, Express, Node.js
  • Responsive to different screen sizes
  • Uses external API - XML feed
  • Deployed on Heroku

    • Next Step - Mobile App

    I'm using Ionic, a hybrid mobile framework which allows applications written in AngularJS, to deploy this web application as a mobile app for IOS and Android. Ionic wraps around Cordova and provides a UI library that makes the app look like it was natively built. This Ionic app will be completed in January 2016.

    User Stories/ Wireframe

    The target audience is people with limited mobility, seniors, those with strollers, or anyone who just doesn't feel like taking the stairs! The app will help someone who can't walk up multiple flights of stairs - they will know to avoid the station before s/he has struggled up entry stairs, paid their fare, only to encounter a broken escalator/elevator!
